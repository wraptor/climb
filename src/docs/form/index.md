## 表单文档

### Attributes

| 参数        | 说明 | 类型 | 可选值 | 默认值 |
| ----------- | ---- | ---- | ------ | ------ |
| option      | 配置参数 |  Object  |        |        |
| v-model/modelValue  | 双向绑定值 |Object   |        |        |

### Events

| 事件名称     | 说明 | 回调参数 |
| ----------- | ---- | ---- |
| submit      |  点击提交触发，需执行done取消loading状态  |  function(form,done) |
